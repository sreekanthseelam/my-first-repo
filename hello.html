Git Basics

Git is a version control system that helps you track changes in your code and collaborate with others. Here are the basic steps to get started:

Git Workflow

  1. Initialize a repository: git init
  2. Check repository status: git status
  3. Add changes to staging: git add .
  4. Commit changes: git commit -m "Your commit message"
  5. Push changes to a remote repository: git push origin main

Common Git Commands

  • Clone a repository: git clone <repository-url>
  • Create a new branch: git branch <branch-name>
  • Switch to a branch: git checkout <branch-name>
  • Merge branches: git merge <branch-name>
  • View commit history: git log
